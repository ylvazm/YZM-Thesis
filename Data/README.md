> This folder contains the cleaned datasets used in the analyses for my thesis.
- Use this data to run steps 4 to 7 of the analyses.
- Running `04_Descriptives.R` will result in the creation of a data file containing the descriptive statistics for the full sample, and a data.frame containing only the omission statistics used in plots later on, in this folder.
- Running `05_Subsets.R` will result in the creation of four subsets sampled from the full datasets, and a data file containing the descriptive statistics for the subsample in this folder.
